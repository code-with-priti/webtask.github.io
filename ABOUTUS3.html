How does an online car park booking system work?

The system enables employees to view available spaces for the specific date / time, reserve a parking slot, and receive the confirmation- all via a mobile app or web application. Whenever there are cancellations or the supposed user of a slot did not show up, the reservation will be automatically released and the parking space will be put back on the available list for reservation. Companies can set a limited time window on the hours that an employee can use the slot, example: each space can be open for morning shift, mid-shift and evening shift schedules separately, and the previous user should make sure to vacate the slot, before the next user arrives.
